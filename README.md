# Mono Software L2 Frontend zadatak

Za pokretanje projekta lokalno:

1. Instaliraj dependencies:
```bash
pnpm i
```
2. Stvori datoteku pod imenom `.env.local` i zalijepi sljedeÄ‡e:

```bash

```
